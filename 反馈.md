# GridCell 底部重新分层 — 完成反馈

## 问题

B级天象(`mt-auto`)、潮汐(`absolute bottom-0.5`)、事件彩条(`absolute bottom-1`) 三者在同一个 bottom 区域重叠。

## 修复

统一底部行为 `absolute bottom-0 left-1 right-1` 的 flexbox 容器，内部垂直排列：

```
┌─ 日期数字 ──────────────────┐
│                              │
│  [S/A圆点 右上角]            │
│                              │
│  底部行 ──────────────────   │
│   B级天象文字 (9px 琥珀色)   │
│   潮汐标签   (9px 红/琥珀)  │
│   ██ 事件彩条 (0.5px)      │
└──────────────────────────────┘
```

- 移除独立的 `AstronomicalMarkers`（B部分）、`TideIndicator` 组件
- 新增 `TopBadge` 组件（仅 S/A 圆点）
- 底部行三个元素垂直 `flex-col gap-px`，不重叠
