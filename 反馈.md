### 新增功能

**布局**：`◀◀ ◀ 0年01月 ▶ ▶▶` 居中对齐

- **◀◀** (ChevronsLeft)：跳至上一年，year=0 时 disabled
- **◀** (ChevronLeft)：上一月，year=0 && month=1 时 disabled（原有逻辑）
- **▶** (ChevronRight)：下一月，year=1200 && month=12 时 disabled（原有逻辑）
- **▶▶** (ChevronsRight)：跳至下一年，year=1200 时 disabled

**标题点击跳转**：
- 点击 "0年01月" → 展开内联输入框：年份 Input + 月份下拉 select + "跳转" 按钮
- 输入完成后点击"跳转"或按 Enter 键 → `goToYearMonth(year, month)`
- ESC 键取消编辑，恢复标题显示
- 输入值自动 clamp 到有效范围（year: 0-1200, month: 1-12）
