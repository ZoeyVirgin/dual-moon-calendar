### 权限分层逻辑

```
游客模式（默认）
├── 可查看日历 + 天象 + 事件列表
├── 无"创建新事件"按钮
└── EventCard 无编辑/删除图标

作者模式（口令验证后）
├── "创建新事件"按钮可见
├── EventCard 显示编辑/删除图标
├── CRUD 操作同时写入 API + localStorage
└── 刷新页面后 sessionStorage 保留登录态
```
