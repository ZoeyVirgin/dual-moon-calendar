# 最终方案 — Framework Other + vercel.json

## 根因

Vite 预设**完全不部署** api/ 目录。连零依赖 `/api/ping` 都 404。

## 方案

```json
{ "buildCommand": "npm run build", "outputDirectory": "dist" }
```

Dashboard → Settings → Framework Preset → **Other** → Save → **手动 Redeploy**

## 验证

| 命令 | 结果 |
|:-----|:-----|
| `npm run build` | ✓ 2.36s |

部署后访问：
- `/api/ping` → 期望 `{"ok":true}`
- `/api/auth` → 期望 200 (口令验证)
