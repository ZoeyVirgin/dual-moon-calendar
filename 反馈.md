# 四合一 — 完成反馈

## Bug 1: GridCell 底部间距

`gap-px` → `gap-0.5`，B级天象文字和潮汐标签有明显间距，不与事件彩条混淆。

## Bug 2: EventModal 原生 select → popover

月份和日期选择器从原生 `<select>` 改为 popover 按钮（ChevronDown 下拉列表 + 自定义滚动条），与 Navigation 风格统一。

添加 `e.type === 'historical-event'` 过滤，周期性节日不计入时间线。
