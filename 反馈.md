# 时间线视图 — 完成反馈

## 新增

`src/components/events/Timeline.tsx` — 垂直事件时间轴

## 修改

`src/App.tsx` — CalendarGrid 与 DetailPanel 之间插入 `<Timeline />`

## 功能

- 按 ABS 升序排列所有可见事件
- 默认折叠（标题栏 "📅 时间线 · N个事件"），点击展开/收起
- 400ms max-height 过渡动画
- 展开状态持久化（`localStorage`）
- 左侧竖线 + 彩色圆点，右侧事件卡片
- 卡片内容：日期 + 类型标签 + 标题 + 描述（≤50字截断）
- 点击卡片 → 跳转到事件日期 + 展开详情面板
- 无事件时自动隐藏
