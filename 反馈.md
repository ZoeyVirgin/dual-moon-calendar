# 紧急修复 — 移除 builds，改为最简 functions

## 问题

上次的 `builds` + `@vercel/static` 配置破坏了静态站点部署，主页打不开。

## 修复

vercel.json 移除 `builds` 和 `routes` 字段，仅保留 `functions`：

```json
{ "functions": { "api/ping.ts": { "runtime": "@vercel/node@3" }, ... } }
```

Dashboard Framework Preset = **Other**（保持）。

## 验证

| 命令 | 结果 |
|:-----|:-----|
| `npm run build` | ✓ 2.49s |

部署后验证：
- 主页 → 正常显示
- `/api/ping` → `{"ok":true}`
