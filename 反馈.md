# vercel.json 最终修复 — 显式 functions 声明

## 修改

- 移除 `framework` 字段（防止 Vercel 检测 vite.config.ts → 套用 Vite 纯静态预设 → 跳过 api/）
- 移除 `rewrites` 字段（Vercel 对 `outputDirectory: "dist"` 自动处理 SPA fallback）
- 添加 `functions` 逐个声明三个 API 路由文件

```json
{
  "functions": {
    "api/auth/route.ts":       { "runtime": "nodejs20.x" },
    "api/events/route.ts":     { "runtime": "nodejs20.x" },
    "api/events/[id]/route.ts":{ "runtime": "nodejs20.x" }
  }
}
```

## 同步手动操作

Vercel Dashboard → 项目 Settings → Build & Development Settings → Framework Preset → **Other** → Save → Redeploy

## 验证

| 命令 | 结果 |
|:-----|:-----|
| `npm run build` | ✓ 2.39s |

push + Dashboard 设 Framework Preset 为 Other + Redeploy 后，`/api/auth` 应返回 200。
