# vercel.json 最终修复 — 完成反馈

## 修改

```json
{
  "rewrites": [
    { "source": "/((?!api/).*)", "destination": "/index.html" }
  ]
}
```

`((?!api/).*)` = 正则负向前瞻，匹配不以 `api/` 开头的所有路径。

| 请求路径 | 行为 |
|:---------|:-----|
| `/` | → rewrite /index.html (SPA) |
| `/app/some-route` | → rewrite /index.html (SPA) |
| `/api/auth` | ✗ 不匹配 rewrite → Serverless Function 处理 |
| `/api/events?abs=0` | ✗ 不匹配 rewrite → Serverless Function 处理 |
